# Proteus Feedback

Bug reports, feature requests, and feedback for **Proteus** -- the automated demo capture and interactive playback tool by [Athena AI](https://athena-ai.co.uk).

## How to Use

1. **Check existing issues** to see if your problem or idea has already been reported.
2. **Open a new issue** using one of the templates below:
   - [Bug Report](https://github.com/AtheneumAI/Proteus-Feedback/issues/new?template=bug_report.yml) -- something is not working
   - [Feature Request](https://github.com/AtheneumAI/Proteus-Feedback/issues/new?template=feature_request.yml) -- suggest a new feature or improvement
   - [General Feedback](https://github.com/AtheneumAI/Proteus-Feedback/issues/new?template=general_feedback.yml) -- anything else

## About Proteus

Proteus is an automated demo capture and interactive playback tool built by Athena AI. Capture high-res screenshots of web applications, detect clickable elements, and produce interactive embeddable demos.
